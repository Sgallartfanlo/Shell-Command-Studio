# Shell Command Studio

> Guia interactiva de comandes per a Bash i PowerShell.

Consulta, busca i copia comandes essencials de Bash i PowerShell, organitzades per categories amb exemples prĂ ctics.

## Tecnologies

- HTML + CSS + JavaScript (vanilla)
- [Tailwind CSS](https://tailwindcss.com/) (via CDN)
- [Font Awesome](https://fontawesome.com/) (icons)
- [Fira Code](https://github.com/tonsky/FiraCode) + [Inter](https://rsms.me/inter/) fonts

## Funcionalitats

- Cerca de comandes per nom, categoria o descripciĂł
- Filtre per categories
- VisualitzaciĂł d'exemples complets
- Copiar comanda al portapapers
- InterfĂ­cie tipus terminal amb estil fosc
- Canvi rĂ pid entre Bash i PowerShell

## Comandes destacades

**Bash**: `ls`, `grep`, `find`, `sed`, `awk`, `ssh`, `rsync`, `iptables`, `systemctl`, `crontab`

**PowerShell**: `Get-ChildItem`, `Select-String`, `Get-Process`, `Invoke-Command`, `Get-WmiObject`, `Get-NetIPAddress`, `Export-Csv`, `Compress-Archive`

## Ăšs

Obre `shell-command-studio.html` al navegador, selecciona Bash o PowerShell, busca per categoria o paraula clau, i fes clic a qualsevol comanda per veure l'exemple complet.
